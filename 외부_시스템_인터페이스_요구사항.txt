2) 외부 시스템 인터페이스 요구사항

[데이터베이스 인터페이스]
IR-EXT-001: MySQL 데이터베이스와의 연동을 위해 JDBC 드라이버를 사용해야 한다.
IR-EXT-002: 데이터베이스는 localhost:3306 포트를 통해 접근 가능해야 한다.
IR-EXT-003: 데이터베이스 연결 시 UTF-8 문자 인코딩을 지원해야 한다.
IR-EXT-004: JPA/Hibernate를 통한 데이터베이스 스키마 자동 관리(DDL auto-update)를 지원해야 한다.

[Google Cloud Vision API 인터페이스]
IR-EXT-005: Google Cloud Vision API v1을 통해 영수증 이미지의 OCR 텍스트 추출을 수행해야 한다.
IR-EXT-006: Vision API 인증을 위해 서비스 계정 키 파일(JSON 형식)을 사용해야 한다.
IR-EXT-007: Vision API 호출 시 OAuth 2.0 인증 프로토콜을 준수해야 한다.
IR-EXT-008: Vision API 응답은 TEXT_DETECTION 기능을 사용하여 텍스트 및 좌표 정보를 포함해야 한다.

[Google Gemini API 인터페이스]
IR-EXT-009: Google Gemini API (gemini-2.0-flash-exp 모델)를 통해 생성형 AI 기반 레시피 생성을 수행해야 한다.
IR-EXT-010: Gemini API 호출은 REST API 방식으로 generativelanguage.googleapis.com 도메인에 접근해야 한다.
IR-EXT-011: Gemini API 인증을 위해 API 키 기반 인증 방식을 사용해야 한다.
IR-EXT-012: Gemini API 응답은 JSON 형식으로 파싱 가능해야 하며, 레시피 정보(재료, 조리 단계)를 포함해야 한다.

[Google OAuth 2.0 인터페이스]
IR-EXT-013: 사용자 인증을 위해 Google OAuth 2.0 프로토콜을 지원해야 한다.
IR-EXT-014: Google OAuth Client ID를 통해 프론트엔드와 백엔드 간 인증 토큰을 검증해야 한다.
IR-EXT-015: JWT(JSON Web Token) 방식으로 사용자 세션을 관리해야 한다.

[외부 레시피 웹사이트 크롤링 인터페이스]
IR-EXT-016: 외부 레시피 웹사이트로부터 HTML 데이터를 크롤링하기 위해 HTTP/HTTPS 프로토콜을 사용해야 한다.
IR-EXT-017: 웹 크롤링 시 Jsoup 라이브러리를 사용하여 HTML 파싱을 수행해야 한다.
IR-EXT-018: 크롤링 요청 시 타임아웃은 8초 이내로 제한되어야 한다.
IR-EXT-019: 크롤링된 레시피 상세 정보 조회 시 타임아웃은 5초 이내로 제한되어야 한다.

[프론트엔드 인터페이스]
IR-EXT-020: 프론트엔드와의 통신을 위해 RESTful API 방식을 사용해야 한다.
IR-EXT-021: CORS(Cross-Origin Resource Sharing) 정책으로 localhost:5173, localhost:3000, localhost:5174 포트를 허용해야 한다.
IR-EXT-022: API 응답은 JSON 형식으로 반환되어야 한다.
IR-EXT-023: 파일 업로드는 multipart/form-data 형식을 지원해야 하며, 최대 5MB 크기 제한을 가져야 한다.

[파일 시스템 인터페이스]
IR-EXT-024: 업로드된 이미지 파일은 src/main/resources/img 디렉토리에 저장되어야 한다.
IR-EXT-025: 저장된 이미지는 /api/images 경로를 통해 접근 가능해야 한다.
IR-EXT-026: 서비스 계정 키 파일은 src/main/resources/keys 디렉토리에 저장되어야 하며, 버전 관리 시스템에서 제외되어야 한다.

[비동기 처리 인터페이스]
IR-EXT-027: 로컬 레시피 조회는 비동기 방식으로 처리되며, 타임아웃은 3초로 제한되어야 한다.
IR-EXT-028: 크롤링 기반 레시피 조회는 비동기 방식으로 처리되며, 타임아웃은 12초로 제한되어야 한다.

[보안 인터페이스]
IR-EXT-029: 모든 API 요청은 JWT 토큰 기반 인증을 거쳐야 한다(인증 제외 엔드포인트 제외).
IR-EXT-030: 비밀번호 및 민감 정보는 암호화하여 저장되어야 한다.
IR-EXT-031: Google Cloud 서비스 계정 키는 환경 변수 또는 안전한 경로에 저장되어야 하며, 소스 코드에 노출되어서는 안 된다.

[네트워크 인터페이스]
IR-EXT-032: 백엔드 서버는 HTTP 프로토콜을 통해 8080 포트에서 실행되어야 한다.
IR-EXT-033: 외부 API 호출 시 HTTPS 프로토콜을 사용하여 보안 통신을 수행해야 한다.
